# Java Annotations

> Metadata for Java code

[![Java](https://img.shields.io/badge/Java-5%2B-blue.svg)](https://www.oracle.com/java/)

## Overview

Annotations provide metadata about the program that is not part of the program itself.

## Topics

1. Built-in Annotations (@Override, @Deprecated, etc.)
2. Custom Annotations
3. Meta-Annotations (@Retention, @Target, etc.)
4. Annotation Processing

## Example

```java
@Override
public String toString() {
    return "Example";
}

@Deprecated
public void oldMethod() { }
```

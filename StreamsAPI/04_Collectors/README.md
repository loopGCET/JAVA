# Collectors

Collectors provide common mutable reduction operations for streams.

## Key Collectors

- **toList/toSet** - Collect to collections
- **joining** - Join strings
- **groupingBy** - Group elements
- **partitioningBy** - Partition by predicate
- **counting/summing/averaging** - Statistics
- **toMap** - Collect to Map

For details, see: **[CollectorsExample.java](CollectorsExample.java)**

# Terminal Operations

Terminal operations trigger stream execution and return a result or side-effect.

## Key Operations

- **collect** - Gather to collection
- **forEach** - Process each element
- **reduce** - Combine to single value
- **count** - Count elements
- **anyMatch/allMatch/noneMatch** - Test conditions
- **findFirst/findAny** - Find elements
- **min/max** - Find extremes

For details, see: **[TerminalOperationsExample.java](TerminalOperationsExample.java)**

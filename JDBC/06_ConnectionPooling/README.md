# Connection Pooling

Reuse database connections for better performance.

Connection pooling maintains a pool of database connections that can be reused, avoiding the overhead of creating new connections.

## Popular Libraries

- **HikariCP** - Fast and lightweight
- **Apache DBCP** - Apache Commons connection pool
- **C3P0** - Mature connection pooling

For details, see: **[ConnectionPoolingExample.java](ConnectionPoolingExample.java)**
